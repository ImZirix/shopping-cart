# Shopping Cart App

A simple React shopping cart app with a home page, shop page, and dynamic cart functionality.

## Features

- Fetches products from FakeStore API
- Users can add, remove, and adjust item quantities
- Navigation between home and shop pages
- Displays total items in the cart

   git clone https://github.com/your-username/shopping-cart.git
